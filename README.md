
## 📦 Что нужно установить?

### 1. **Ollama** - система запуска локальных ИИ-моделей
- Скачать: https://ollama.com/
- Просто запустите установщик как обычную программу

## 🚀 Как запустить?

### Шаг 1: Запустите Ollama (один раз)
```cmd
# Откройте командную строку (cmd) и введите:
ollama serve
```
Оставьте это окно открытым!

### Шаг 2: Откройте проект UE5
1. Запустите Unreal Engine 5
2. Откройте наш проект
3. Нажмите Play

## 🛑 Как завершить процессы?

### Остановить Ollama:
```cmd
# Вариант 1 - через диспетчер задач:
1. Ctrl + Shift + Esc
2. Найти "ollama.exe"
3. Снять задачу

# Вариант 2 - через командную строку:
taskkill /f /im ollama.exe
```

### Остановить UE5:
- Просто закройте Unreal Engine как обычную программу

## 🔧 Быстрые команды для cmd

| Команда | Что делает |
|---------|------------|
| `ollama list` | Показывает загруженные модели |
| `ollama run deepseek-r1` | Запускает чат с ИИ в консоли |
| `ollama ps` | Показывает запущенные модели |
| `ollama stop` | Останавливает все модели |

### Если что-то не работает:
1. Проверьте, что Ollama запущен: `ollama list`
2. Проверьте порт: браузер → `http://localhost:11434`
3. Перезагрузите модель: `ollama restart deepseek-r1`

## 💡 Советы

### Производительность:
- Нужно минимум **16GB оперативной памяти**
- Первый запуск модели будет долгим (~5 минут)
- Последующие запуски быстрые

### Проект UE5:
- Открывайте файл `*.uproject`
- Нажмите кнопку Play в редакторе
- Аватар (голова) покажет ответы ИИ на экране

### Тестирование:
```cmd
# Проверьте что ИИ работает:
curl http://localhost:11434/api/generate -d "{\"model\":\"deepseek-r1\",\"prompt\":\"Привет\"}"
```

## ❓ Частые проблемы


### "UE5 не видит Ollama"
1. Убедитесь что `ollama serve` запущен
2. Проверьте в браузере: `http://localhost:11434`
3. Перезапустите UE5
